# Gjenlevende BS Sak

Saksbehandlingsløsning for gjenlevende barnetilsyn og skolepenger

## Kom i gang

### Installasjon

Install the dependencies:

```
npm install
```

### Hent og sett miljøvariabler

Kjør shell scriptet hent-og-lagre-miljovariabler

```
sh hent-og-lagre-miljovariabler.sh
```

### Development

Start the development server with HMR:

```
npm run dev
```

Gå til http://localhost:8080/
